# Bank Application

The aim of this simple application is to handle the bank account opening and requests of new customers.


## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software and how to install them

```
Java (JDK8+)
```

### Setup

Copy and paste the following command in your command line
```
git clone https://github.com/iTomsteR/Bank_App.git
```
or download the project directory.

#### Run from command line
Open command line and `cd` to the project directory.
Type the following command in order to compile and execute the program:
```
./CompileProgram.sh
```
#### Run from IDE
If you want to run the project from an IDE of your choice:
1. Import the project directory
2. Run

### Features
* Gather user data from CSV file
* Provide 2 account types
* Generate account number
* Display account summary containing all relevant information
